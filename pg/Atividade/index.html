Projeto interativo

Este projeto apresenta uma interface web responsiva que permite ao usuário alternar entre os temas Claro (Light), Escuro (Dark) e Original em tempo real. A troca de tema é feita inteiramente em JavaScript, aplicando classes CSS ao elemento , garantindo que todo o layout se adapte instantaneamente sem alterar imagens ou outros elementos visuais importantes.
